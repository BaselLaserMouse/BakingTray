How to use the simulated ScanImage machine data file.

1) In NI MAX create the following simulated devices:
PXIe-6115 (rename it "scan")
PXIe-6341 (rename it "aux1")
PCIe-6343 (rename it "pmt")

2) Copy "Machine_Data_File_LinScan_SIMULATED.m" into your ScanImage install directory

3) Start ScanImage and open the above MDF
